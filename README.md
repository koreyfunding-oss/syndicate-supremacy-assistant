# Syndicate Supremacy Assistant  

Web edition of Syndicate Supremacy poker/blackjack assistant with referral unlock system, neon-cyan UI, and CIS integration.  

## Features  
- Custom game logic for poker and blackjack with secure state management.  
- Referral-based unlock system for free tier and sharing incentives.  
- Integration with CIS for authentication and user management.  
- Deployment-ready configuration for Vercel.  
- Anti-copycat security measures including code obfuscation and rate limiting.  
- Neon-cyan themed UI built using Figma prototypes.  

## Setup  

Initial project scaffold will include Next.js or similar framework, serverless functions for game logic, and database integration via Supabase/Firebase. Further code will be added in subsequent commits.
